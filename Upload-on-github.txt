sab sa phale kya kana ha apko 
----------------------------
1) vite.config ma 
plugins: [react()], // iske neeche ayaga
base: '/your-repo-name/', // ⚠️ repo name daalo yahan

2) Install gh-pages
npm install gh-pages --save-dev

3)package.json me scripts add karo:
"scripts": {
  "dev": "vite",
  "build": "vite build",
  "preview": "vite preview",
  "deploy": "gh-pages -d dist"
}

4)Deploy Steps

npm run build     # production build banayega dist/ folder me
npm run deploy    # gh-pages par push karega

Bas GitHub par ja kar gh-pages kardo 
Done!
